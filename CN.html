computer components

Central Processing Unit (CPU): Often called the brain of the computer, the CPU performs calculations and executes instructions.
Motherboard: The main circuit board that connects and allows communication between all the computer's components.
Random Access Memory (RAM): Temporary memory used by the computer to store data and instructions that are actively being used or processed.
Storage (Hard Drive or SSD): The component that stores all the data, files, and applications, including the operating system, either on a hard drive (HDD) or a solid-state drive (SSD).
Power Supply Unit (PSU): Converts electrical power from an outlet to usable power for the computer's components.
Graphics Processing Unit (GPU): Handles rendering images, videos, and animations, important for gaming and visual computing.
Input Devices: Includes keyboards, mice, and other peripherals that allow users to interact with the computer.
Output Devices: Monitors, printers, and speakers that display or convey the results of the computer's processing.
